Weather Station Test

Installation / Usage

  • There is no specific server-side installation required.
  • You can run the files from your local drive or from any HTTP server.
  • The whole app is pure Javascript/HTML.
  • It requires an internet connection to access the weather API
  • To generate a release build, you need node.js to run r.js (the optimizer) with buid/app.build.js:
    cd js
node lib/r.js -o ../build/app.build.js

Implementation notes

  • Since this was a programming test I didn't use jQuery or any other framework (except RequireJS).
  • Unit tests are done with QUnit and can be accessed here.
  • It supports multiple weather data per location in the full view: it will render the list of icons
  • To support IE7, I would have to write a querySelector() failover or use jQuery

Possible improvements

  • Support IE7
  • More fine-tuning for specific mobile phones
